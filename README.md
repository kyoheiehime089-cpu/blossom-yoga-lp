# csv-member-manager

小規模店舗向けの会費CSV・売上・会員管理自動化ツールです。

## 目的

毎月の会費CSVをスプレッドシートに貼るだけで、以下を自動で確認できるようにします。

- 月別売上
- 会員数
- コース別人数
- 未入金
- 金額ミス
- 前月差分
- 月次レポート

## 最初の商品

**会費CSV自動集計シート**

最初はGoogleスプレッドシートとGoogle Apps Scriptで作ります。
いきなり本格的な予約システムは作らず、毎月のCSV集計を楽にする小さな商品から始めます。

## 進め方

1. サンプルCSVを作る
2. Apps Scriptで集計機能を作る

4. 使い方マニュアルを作る
5. 販売資料を作る
6. モニター価格で1件目を取りに行く
